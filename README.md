# hibernate-postgresql-persistence-example-configuration

Example configuration of `persistence.xml` in Maven.

## Maven

`persistence.xml`file should be kept as in structure below.
```bash
.
├── ...
├── src                    
|   ├── main          
|       ├── ...
|       └── resources
|           └── META-INF
|               └── persistence.xml              
└── ...
```
